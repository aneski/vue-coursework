## About

This is a coursework project featuring a full-styled front-end application built with Vue 3 and Vite. The application allows students and their parents to:
- Browse a variety of after-school classes (music, STEM, sports, arts, etc.)
- Search and filter classes by category
- Add classes to a shopping cart
- Complete checkout with student and parent information
- View order confirmations

## Features

- **Home Page**: Beautiful landing page with featured classes and key benefits
- **Class Catalog**: Browse all available classes with detailed information
- **Search & Filter**: Find classes by name or filter by category
- **Shopping Cart**: Add multiple classes, adjust quantities, and review your selections
- **Checkout**: Complete order form for student and parent details
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## Technologies Used

- Vue 3 with Composition API
- Vite for fast development and building
- Modern CSS with gradients and animations
